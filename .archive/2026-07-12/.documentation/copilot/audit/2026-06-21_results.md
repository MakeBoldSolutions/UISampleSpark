## Recommendations (Priority Order)

### 🟠 HIGH — Fix Immediately

**ARCH1** — Add `.ConfigureAwait(false)` to `EmployeeDB.cs`

`UISampleSpark.Core/Repository/EmployeeDB.cs` has 10+ async calls in the repository layer
that do not call `.ConfigureAwait(false)`. This is a Principle II MUST requirement.

Each affected line should change from:
```csharp
await _context.Employees.FindAsync(ID);
```
to:
```csharp
await _context.Employees.FindAsync(ID).ConfigureAwait(false);
```

Affected lines: 79, 87, 100, 107, 119, 125, 144, 145, 150, 163 (and any others in the file).

---

### 🟡 MEDIUM — Fix in Next Cycle

**QUAL1** — Add `ImplicitUsings` to `UISampleSpark.Core.Tests.csproj`

```xml
<PropertyGroup>
  <ImplicitUsings>enable</ImplicitUsings>
</PropertyGroup>
```

Also create `UISampleSpark.Core.Tests/GlobalUsings.cs` with project-level usings,
consistent with all other projects in the solution.
